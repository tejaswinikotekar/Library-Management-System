# Library Management System

A full-stack Library Management System built using React.js, Spring Boot, and MySQL.

## Features
- User Registration & Login
- Admin Dashboard
- Book Management
- Book Issue & Return
- Borrowed Books Tracking
- REST APIs
- MySQL Integration

## Tech Stack
- Java 17
- Spring Boot
- React.js
- MySQL
- Spring Data JPA
- Maven
- Axios
